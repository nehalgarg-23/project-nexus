**Features:**
* **Customer Side (customerSide Folder):** Stores the website and allows customers to:
    * Make reservations
    * Register for accounts
    * View profile points


**Steps to run the project locally for Netbeans Manually:**

1. Download and install XAMPP in your machine
2. Clone or download the repository
3. Extract all the files and move it to the 'htdocs' folder of your XAMPP directory and name it as restaurant.
4. Start the Apache and Mysql in your XAMPP control panel.
5. Open your web browser and type 'localhost/phpmyadmin'
6. In phpmyadmin page, create a new database from the left panel and name it as 'restaurant'
7. Import the file 'restaurant.sql' inside your newly created database and click ok.
8. Open a new tab and type 'localhost/restaurant' in the url of your browser.
9. hurray thats it.
